PCA for Image Classification
(from Scratch)
(Spring 2023)

Implemented PCA to reduce dimensionality, visualized PCs, determined optimal subspace dim from eigenvals, calculated error rates, and ID'd least 5-like image w/ orthogonal energy analysis.

MNIST Nearest Neighbors
(from Scratch)
(Spring 2023)

Using database with 5000 training images, and 500 test images of hand-written numerals, I trained a NN classifier. I utilized the Euclidean distance metric and tested it on the test database.

ProstheTech
(HARD Hack Runner-Up)
(Spring 2023)

Our device includes a 3D-printed EMG that attaches to the user's arm and wirelessly transmits flexion data to a Raspberry Pi-controlled camera while displaying real-time EMG data and camera feed on a web server.

Hate Speech NLP Analysis
(Fall 2022)

Detecting and analyzing hate speech on videos relating to sexism on a right-wing platform (NLTK, scikit-learn, pandas).

California Wage Data Analysis
(Fall 2022)

California employment and wage analysis on data from the past decade. Libraries utilized include Pandas, Matplotlib, pyMannkendal, and Seaborn.

Drowsiness Detection with OpenCV and Arduino
(Spring 2021)

In this project, a colleague and I programmed a software that detects when the user becomes drowsy before alerting them with a buzzing noise until they open their eyes.

Non-Contact Range Detection of HCSR04 Sensor
(Fall 2020)

My team used MATLAB, an Arduino MEGA, LEDs, a buzzer, an IR remote and receiver, and a DHT11 temperature sensor to create a security device.

Portfolio Site
(Summer 2020)

Using HTML, CSS, and JavaScript, I created this portfolio site to display my projects, education, and contact information.

Tensile Strength of ABS Plastic Moldings and 3D Printed Samples
(Summer 2019)

At SDSU, I researched the tensile strength of 3D printed ABS ASTM D638 samples versus ASTM D638 samples created with injection molding (~100 hours).
